# 🎓 Grade Calculator (Python)

## 📌 Description
This is a modular Grade Calculator program written in Python. It collects marks of multiple students, calculates their average, assigns grades, and displays a summary report.

## ⚙️ Features
- Input marks for multiple students
- Calculate average marks
- Assign grades (A, B, C, etc.)
- Clean and simple output
- Modular code using functions

## 📂 Project Structure
- grade_calculator.py  → Main program
- utils.py             → Helper functions

## ▶️ How to Run
1. Open the project folder in VS Code
2. Run the main file:
   python grade_calculator.py

## 💡 Concepts Used
- Functions
- Loops
- Lists & Dictionaries
- Modular Programming
